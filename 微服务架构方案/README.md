
# 微服务架构方案

   微服务架构一般会从两个模式中演化:
    
    1.  一种就是一开始就是整体的应用，所有的模块都是紧耦合的。单体系统时代
    
    2.  另外一种是面向服务的架构模式(SOA，service-oriented architecture pattern)。SOA 时代
